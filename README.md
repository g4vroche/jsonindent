# jsonindent
Pretty json output for CLI



## Usage

<code lang="sh">
$ cat file.json | ji
</code>


<code lang="sh">
$ curl -s 'http://api.example.com/unreadable-response.json' | ji
</code>



## Instalation 
<code lang="sh">
ln -s /path/to/this/repo /usr/bin/ji
</code>
